mbnntp - News server

Synopsis.

mbnntpYou should not run this from the commandline, instead it should be run by inetd or xinetd.

 

Description.

mbnntp is a RFC977 and RFC2980 compatible news server. It uses the bbs echomail and news areas. To use the server, the user must login in the news server with his bbs unix name and password, thus the user must exist on the bbs. The access to the areas is the same as if the user would be logged into the bbs. Only echomail areas are available to the user as newsgroups.

 

Installation.

If installed from inetd add the following line to /etc/inetd.conf: 

nntp    stream  tcp     nowait  mbse    /opt/mbse/bin/mbnntp    mbnntp

 

Bugs.

Maybe a lot, this is a new program.
